Examples

Which is better, Documented CSS or External CSS? Why?

External CSS is better, because it is applied to many pages instead of just one. it can be imported from other web projects, can be stored anywhere. and modification is easier since it is done in one page.

Should we avoid inline CSS? Why?

Yes, because it is time consuming to style each element individually

Give an example of a:

Universal Selector: 

		* {
			margin: 0;
			padding: 0;
		  }

Attribute Selector: 

		a[title] {
			color: purple;
		  }

Pseudo Classes: 

		button:hover {
			color: blue;
		  }
